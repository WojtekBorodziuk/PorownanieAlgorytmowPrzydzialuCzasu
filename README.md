# PorownanieAlgorytmowPrzydzialuCzasu
Projekt symulacyjny porównujący wydajność trzech klasycznych algorytmów szeregowania procesów w systemach operacyjnych:
(FCFS - First Come First Served, SJF - Shortest Job First, LCFS - Last Come First Served)

Celem projektu jest analiza efektywności algorytmów przydziału czasu procesora poprzez obliczenie średnich czasów oczekiwania oraz cyklu przetwarzania dla zestawów losowych procesów.
